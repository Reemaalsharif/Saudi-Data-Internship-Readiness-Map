# Saudi Data Internship Readiness Map

This project analyzes Saudi data-related job postings to identify the most requested skills, tools, and focus areas, then transforms these insights into a practical readiness roadmap for Computer Science students interested in Data Science and analytics careers.

---

## Project Goal

The goal of this project is to understand the requirements of Saudi data-related roles and translate real job market insights into a structured learning roadmap that helps Computer Science students prepare for data internships and early-career opportunities.

---

## Main Question

**What skills, tools, and knowledge areas should a Computer Science student in Saudi Arabia develop to become ready for a data-related internship?**

---

## Research Questions

This project explores:

- What are the most required technical skills in Saudi data-related roles?
- Which tools appear most frequently, such as SQL, Python, Power BI, Excel, and Tableau?
- How do requirements differ between Data Analyst, Data Scientist, BI, AI, and other analytics-related roles?
- Which cities show higher demand for data-related opportunities?
- What learning path should students follow to become internship-ready?
- What skills separate beginner-level preparation from career-ready capabilities?

# Dataset

The final cleaned dataset contains **45 Saudi data-related job postings** collected from different data career categories.

The dataset includes roles related to:

- Data Analysis
- Data Science
- Business Intelligence
- AI-related roles
- Data Governance
- Analytics and reporting roles
- Internship and early-career opportunities

The cleaned dataset includes additional analysis-ready columns:

- `city_cleaned`
- `experience_category`
- `role_category`

These columns improve the analysis by allowing comparisons between different cities, experience levels, and job categories.

# Career Readiness Roadmap

Based on the Saudi data job market analysis, the project developed a structured roadmap for Computer Science students preparing for data careers.

## Beginner Level

Focus areas:

- Python Basics
- SQL Fundamentals
- Excel
- Data Cleaning
- Data Analysis
- Visualization Basics

## Intermediate Level

Focus areas:

- Advanced SQL
- Statistics
- Data Modeling
- Machine Learning Fundamentals
- Advanced Power BI
- Cloud Data Concepts

## Career Ready Level

Focus areas:

- Building Real Projects
- GitHub Portfolio
- CV Preparation
- Interview Preparation
